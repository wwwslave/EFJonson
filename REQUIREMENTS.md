# Requisitos para Aprender Git y GitHub

A continuación, se detallan los requisitos recomendados para estudiantes que deseen aprender a usar Git y GitHub efectivamente.

## 🖥️ Hardware y Sistema Operativo

- [ ] **Computadora** con las siguientes características mínimas:
  - Procesador de 1 GHz o superior
  - 4 GB de RAM (8 GB recomendado)
  - 5 GB de espacio libre en disco
- [ ] **Sistema Operativo** compatible:
  - Windows 10/11 (64-bit)
  - macOS 10.14 (Mojave) o superior
  - Linux (Ubuntu 20.04+, Fedora, Debian, etc.)

## 📚 Conocimientos Previos

- [ ] Conocimiento básico de **línea de comandos/terminal**:
  - Navegación entre directorios (`cd`, `ls`, `pwd`)
  - Creación/eliminación de archivos y carpetas
- [ ] Familiaridad con **conceptos básicos de programación**:
  - Saber qué son archivos de código y directorios
  - Entender qué es un "proyecto de software"
- [ ] Nivel básico de **inglés técnico** (la mayoría de la documentación está en inglés)

## 🛠️ Software Requerido

### Git y Herramientas Relacionadas
- [ ] **Git** instalado:
  - Windows: [Git for Windows](https://git-scm.com/download/win)
  - macOS: `brew install git` o descargar desde [git-scm.com](https://git-scm.com/download/mac)
  - Linux: `sudo apt install git` (Ubuntu/Debian) o `sudo dnf install git` (Fedora)
- [ ] **Editor de código** (opciones recomendadas):
  - [Visual Studio Code](https://code.visualstudio.com/) (Recomendado)
  - [Atom](https://atom.io/)
  - [Sublime Text](https://www.sublimetext.com/)
- [ ] **GitHub Desktop** (opcional para principiantes): [Descargar aquí](https://desktop.github.com/)

### Navegador Web
- [ ] Navegador moderno instalado:
  - Google Chrome
  - Firefox
  - Microsoft Edge
  - Safari (macOS)

## 🌐 Cuentas Necesarias
- [ ] Cuenta en **GitHub**: [Registrarse aquí](https://github.com/signup)
- [ ] (Opcional) Cuenta en **GitLab** o **Bitbucket** para explorar otras plataformas

## 📌 Recomendaciones Adicionales
- [ ] Conexión a Internet estable (para interactuar con repositorios remotos)
- [ ] (Opcional) Cliente GUI para Git como:
  - Sourcetree
  - GitKraken
  - Fork
- [ ] (Opcional) Conocimiento básico de Markdown para documentación
